Как получить API-ключ
Зарегистрируйтесь на exchangerate-api.com.
Создайте аккаунт и получите бесплатный API-ключ.
Вставьте ключ в переменную API_KEY в файле скрипта.
Использование:
Запустите скрипт.
Выберите валюты "из" и "в".
Введите сумму.
Нажмите "Конвертировать".
История отображается в таблице.
Примеры использования:
Конвертация различных валют.
Проверка некорректных данных (отрицательная сумма, пустое поле).
